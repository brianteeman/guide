# Technical Requirements

## Server Requirements <a id="server-requirements"></a>

Technical requirements for J2Store:

* PHP 5.5 or greater
* MySQL 5 or greater
* Joomla 3.3.x or greater
* CURL and JSON modules enabled

Thats it !
